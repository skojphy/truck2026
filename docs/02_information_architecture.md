# 02. Information Architecture

## Sitemap

1. **Home (Platform Hub)**
    - Hero: Featured Life Modes
    - Sections: Today's Mode, Popular Items, Best Creators
2. **Life Mode Discovery**
    - Category-based mode browsing
    - Personalized recommendations
3. **Mode Detail**
    - Experience content (Tips, Guides)
    - Product Checklist & Grid
4. **AI Mode Assistant**
    - Natural language companion for shopping & planning
5. **Community**
    - User reviews and lifestyle sharing

## User Flow

Explore Mode → Learn & Get Inspired → View Related Products → Purchase (Platform Result)
