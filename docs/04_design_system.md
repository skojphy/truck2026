# 04. Design System

## Typography

- **Primary Font**: Gmarket Sans
- **Scale**:
    - Display: 48px ~ 64px
    - Heading: 24px ~ 32px
    - Body: 14px ~ 16px

## Colors

- **Heritage Blue**: #2E3192 (Platform Base)
- **Heritage Green**: #00B31E (Point & Growth)
- **Surface**: #FFFFFF / #F8F9FA
- **Text**: #1A1A2E (Primary) / #5A5A7A (Secondary)

## Components

- **Premium Card**: Glassmorphism applied with soft shadows
- **Product Grid**: Scale-focused dense layout
