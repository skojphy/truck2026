# 03. Feature Specification

## 1. AI Mode Assistant

- 대형 언어 모델 기반의 라이프스타일 가이드
- 질문에 따른 맞춤형 체크리스트 자동 생성
- 장바구니 일괄 담기 기능 제공

## 2. Mode Curation

- 전문 크리에이터와 지마켓 MD가 협업한 '모드 패키지'
- 상품 간의 맥락적 연결 (예: 캠핑 텐트 + 어울리는 조명 + 간편식)

## 3. Platform Mesh Navigation

- 수많은 카테고리를 직관적으로 연결하는 메가 메뉴
- 실시간 인기 모드 및 상품 랭킹 시스템
